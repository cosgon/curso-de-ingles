# Lesson 19 – Comparison: Going to vs Will

**Objetivo:** Diferenciar planos (going to) de decisões espontâneas/previsões (will).

## Verbos
- think, believe, promise, offer, decide

## Gramática
- Uso de going to para planos pré-estabelecidos; will para previsões, promessas, ofertas, decisões no momento.

## Preparação
Estudar a diferença.

## Desenvolvimento
1. Warm-up: perguntas sobre planos.
2. Vocabulário: contexto de reuniões.
3. Verbos: praticar com ambos.
4. Gramática: explicar diferenças com exemplos.
5. Prática guiada: escolher a forma correta.
6. Produção livre: role-play de uma reunião de planejamento.
7. Feedback.
